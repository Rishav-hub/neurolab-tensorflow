Requirements -:
- Python 3.8.13
- tensorflow==2.10.0
- matplotlib==3.6.0
### How to Run

- Given all the packages are installed.
- Install some of the dependencies-: 
    ```
    pip install -r requirements.txt
    ```

- tensorflow Example 
    ```
    python tensorflow_example.py
    ```
